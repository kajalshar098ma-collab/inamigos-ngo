# InAmigos Foundation Awareness Webpage

## Overview

This is a simple NGO awareness webpage created using HTML and CSS. The website provides information about InAmigos Foundation, its ongoing projects, social impact, gallery, and contact details.

## Features

* Responsive design
* About NGO section
* Ongoing projects section
* Social impact statistics
* Image gallery
* Contact information
* Volunteer call-to-action

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome Icons

## Project Structure

NGO-Webpage/
│
├── index.html
├── style.css
└── images/

## Purpose

The purpose of this website is to create awareness about the NGO's initiatives and encourage people to support, volunteer, and participate in social welfare activities.
